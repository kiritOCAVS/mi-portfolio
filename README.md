# Mi Portfolio - Desarrollador Full Stack

Bienvenido a mi portfolio. Este proyecto es una presentación de mis habilidades y proyectos realizados.

## Descripción

Este portfolio contiene información sobre:
- **Mis proyectos**: Bots de Discord, Sistema de Gestión Pizzería, Proodos App
- **Mis habilidades**: JavaScript, Node.js, HTML/CSS, Bases de Datos
- **Formas de contacto**: Formulario de contacto y redes sociales

## Proyectos Destacados

### 1. Bots Discord Personalizados
Desarrollo de bots usando Discord.js con funcionalidades como:
- Sistemas de moderación
- Comandos personalizados
- Manejo de eventos
- Integración con bases de datos

**Tecnologías**: JavaScript, Node.js, Discord.js, SQLite/MongoDB

### 2. Simulador de Préstamos Personales
Calculadora de créditos con múltiples funcionalidades:
- Cálculo de préstamos con diferentes planes (12, 24, 36, 48 cuotas)
- Intereses ajustables según el plazo
- Historial de simulaciones guardado en localStorage
- Interfaz intuitiva con validaciones

**Tecnologías**: JavaScript, HTML5, CSS3, LocalStorage

### 3. LyG Motos - E-commerce
E-commerce completo de productos para motocicletas:
- Catálogo de productos por categorías (cubiertas, cascos, accesorios)
- Carrito de compras persistente
- Formulario de checkout
- Órdenes guardadas en Firebase/Firestore
- Sistema de detalle de productos

**Tecnologías**: React, Firebase, React Router, Vite

## Cómo usar

1. **Abre el archivo**: Simplemente abre `index.html` en tu navegador web
2. **No necesita servidor**: Este portfolio es completamente estático
3. **Personaliza**: Edita el HTML con tu información y actualiza los enlaces

## Personalización

Para hacer este portfolio tuyo:

1. **Cambiar datos personales**:
   - Edita el título y descripción en `index.html`
   - Actualiza los enlaces de GitHub y LinkedIn
   - Cambia el email de contacto

2. **Modificar proyectos**:
   - Actualiza la sección de proyectos con tus propios proyectos
   - Añade o quita tarjetas según necesites

3. **Ajustar estilos**:
   - Los colores están definidos en las variables CSS en `styles.css`
   - Puedes cambiar fuentes, tamaños y espacios fácilmente
